统一路由解决方案
---

### 项目结构

```
.
├── TTPlugin
│       └── plugin  // 可独立调试/发布的gradle插件
├── _demoapp        // 测试app
├── _demolib        // 测试lib
├── tt_annotation   // TT注解
├── tt_core         // TT路由分发执行器
└── tt_processor    // TT注解处理器
```

### 构建目标

1. 将路由注册定义工作前置至编译期, 尽量减少路由自身的运行时初始化工作
2. 降低使用者的学习门槛,做到注册自动化
3. 实现综合路由, 完成页面路由、服务抽象、组件路由的核心交互能力
